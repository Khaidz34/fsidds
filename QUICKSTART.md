# ⚡ Quick Start - FSI DDS

## 🚀 Bắt đầu trong 5 phút

### 1️⃣ Mở ứng dụng
```powershell
# Cách 1: Dùng script
.\test.ps1
# Chọn option 2

# Cách 2: Trực tiếp
Start-Process "frontend/index.html"

# Cách 3: Double click
# Double click file frontend/index.html
```

### 2️⃣ Đăng nhập
```
Admin:
  Username: admin
  Password: password

User:
  Username: nhanvien1
  Password: 123456
```

### 3️⃣ Sử dụng
- **User:** Dashboard → Đặt cơm hôm nay → Chọn 2 món → Đặt cơm
- **Admin:** Quản lý Menu → Nhập món → Đăng menu

---

## 📱 Chức năng chính

### 👤 Dành cho User
1. **Đặt cơm:** Chọn 2 món từ menu hôm nay
2. **Xem đơn:** Lịch sử đặt cơm của bạn
3. **Góp ý:** Gửi feedback cải thiện hệ thống

### 👨‍💼 Dành cho Admin
1. **Đăng menu:** Post menu hàng ngày
2. **Xem đơn:** Tất cả đơn hôm nay
3. **Xuất Zalo:** Copy text gửi cho quán
4. **Thống kê:** Xem báo cáo tuần/tháng
5. **Thanh toán:** Quản lý thanh toán
6. **Hóa đơn:** Xuất PDF hóa đơn
7. **Quản lý:** User management
