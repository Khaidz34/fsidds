# FSIDDS

Full-stack application with React frontend and Express backend.

## Frontend
- React + TypeScript + Vite
- Deployed on GitHub Pages: https://khaidz34.github.io/fsidds/

## Backend  
- Express.js + CORS
- Deployed on Render: https://fsidds.onrender.com

## Database
- Supabase PostgreSQL

## Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
