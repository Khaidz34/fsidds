Tại sao chọn bộ ba này? Netlify phục vụ HTML tĩnh với CDN toàn cầu (cực nhanh). Render chạy Node.js API miễn phí. Supabase là PostgreSQL thật sự, không giới hạn số bảng, có sẵn Auth và realtime nếu cần mở rộng sau.
🗄️ SUPABASE — Database

Tạo Database trên Supabase

  1. Vào supabase.com → Sign Up (dùng GitHub hoặc email)
  2. Nhấn "New Project" → Điền tên: fsidds → Chọn region Southeast Asia (Singapore) → Đặt mật khẩu DB → Create
  3. Chờ ~2 phút cho project khởi tạo xong
  4. Vào SQL Editor (menu trái) → Nhấn "New query"
  5. Copy toàn bộ nội dung file schema.sql → Paste vào → Nhấn Run ▶
  6. Vào Settings → API, copy 2 giá trị sau:
📋 Project URL: https://xxxxxxxx.supabase.co
📋 service_role key (không phải anon key): eyJhbGci...
⚠️ Lưu ý: Dùng service_role key (không phải anon), key này có quyền bypass RLS để backend hoạt động đúng.
⚙️ RENDER — Backend API

Deploy Backend lên Render.com

  1. Tạo tài khoản tại render.com (dùng GitHub)
  2. Push thư mục backend lên GitHub repo mới (hoặc dùng Render Deploy trực tiếp)
  3. Trên Render: New → Web Service → Connect GitHub repo của bạn
  4. Cấu hình như sau:
# Cài đặt trên Render Dashboard
Name:          fsidds-backend
Runtime:       Node
Build Command: npm install
Start Command: node server.js
Plan:          Free
  1. Cuộn xuống phần Environment Variables, thêm 4 biến sau:
SUPABASE_URL         = https://xxxxxxxx.supabase.co
SUPABASE_SERVICE_KEY = eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
JWT_SECRET           = fsidds_mat_khau_bi_mat_cua_ban_2024
FRONTEND_URL         = https://fsidds.netlify.app
  1. Nhấn Create Web Service → Chờ deploy ~3 phút
  2. Copy URL của service, dạng: https://fsidds-backend.onrender.com
ℹ️ Free tier Render: Server sẽ "ngủ" sau 15 phút không có request. Lần đầu truy cập sẽ chờ ~30 giây để server thức dậy. Nếu muốn luôn online, dùng UptimeRobot ping 14 phút/lần (cũng free).
🌐 NETLIFY — Frontend

Deploy Frontend lên Netlify

  1. Mở file frontend/index.html trong VSCode hoặc Notepad
  2. Tìm dòng này ở đầu phần <script>:
// ĐỔI URL NÀY THÀNH URL RENDER CỦA BẠN
const API = 'https://fsidds-backend.onrender.com';
  1. Thay URL thành URL Render bạn vừa copy ở bước 2
  2. Vào netlify.com → Sign Up → Chọn "Deploy manually"
  3. Kéo thả thư mục frontend vào ô deploy trên Netlify
  4. Netlify tự tạo URL dạng: random-name.netlify.app
  5. Vào Site settings → Change site name → Đổi thành fsidds (nếu còn trống)
🎉 URL cuối cùng của bạn:
🌐 Frontend: https://fsidds.netlify.app
⚙️ Backend API: https://fsidds-backend.onrender.com
⚙️ CẬP NHẬT CORS

Cập nhật FRONTEND_URL trên Render

  1. Vào Render Dashboard → Service fsidds-backend → Environment
  2. Cập nhật biến FRONTEND_URL thành URL Netlify thật của bạn:
FRONTEND_URL = https://ten-site-cua-ban.netlify.app
  1. Nhấn Save Changes → Render tự động redeploy

✅ Kiểm tra & Đăng nhập lần đầu

  1. Mở URL Netlify của bạn trên trình duyệt
  2. Đăng nhập với tài khoản admin mặc định:
Tên đăng nhập: admin
Mật khẩu:      admin123
  1. Vào Quản lý tài khoản → Tạo tài khoản cho nhân viên
  2. Vào Đăng menu → Nhập danh sách món hôm nay
  3. Chia sẻ URL Netlify cho toàn bộ nhân viên để đăng ký và đặt cơm!
✅ Hệ thống đã sẵn sàng! 100 người có thể truy cập đồng thời. Netlify CDN xử lý frontend, Render chạy API, Supabase quản lý data — tất cả hoàn toàn miễn phí.

💡 Mẹo giữ server Render luôn online (miễn phí)

  1. Vào uptimerobot.com → Tạo tài khoản miễn phí
  2. Thêm monitor mới: HTTP(s) → URL: https://fsidds-backend.onrender.com
  3. Interval: 5 phút → Create Monitor
  4. UptimeRobot sẽ ping server mỗi 5 phút → server không bao giờ ngủ!
